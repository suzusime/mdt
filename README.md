# mdt
an command-line interface for [marked-terminal](https://github.com/mikaelbr/marked-terminal)

## Usage
```bash
mdt README.md
```

```bash
mdt README.md | less -R
```

```bash
cat README.md | mdt
```

## Install
```bash
npm install -g git+https://git@github.com/suzusime/mdt.git
```
